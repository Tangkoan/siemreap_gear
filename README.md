# 🎒 Siem Reap Gear - Final Year Project (Sarana) 🎓

**Siem Reap Gear** is my **Bachelor’s Degree Final Project (Sarana)**. It is a comprehensive E-commerce and Equipment Rental platform designed specifically for the tourism and adventure community in Siem Reap, Cambodia.

## 📝 Project Overview
This project was developed to solve the challenges travelers face when looking for high-quality outdoor gear in Siem Reap. It serves as a bridge between gear providers and adventurers, providing a seamless digital marketplace experience.

## 🛠️ Technical Implementation (Tech Stack)
- **Framework:** [Laravel 11/12](https://laravel.com/) (Robust Backend Logic)
- **Frontend:** Tailwind CSS & Blade Templating (Responsive UI)
- **Database:** MySQL (Relational Data Mapping)
- **Auth:** Custom Role-Based Access Control (Admin & Customer)

## ✨ Key Features
- **Product Inventory:** Dynamic management of gear for sale and rent.
- **Booking & Rental System:** Specialized logic for equipment rental durations and availability.
- **User Dashboard:** Separate interfaces for customers to track orders and admins to manage the shop.
- **Location-Based Focus:** Optimized for the Siem Reap local market needs.

## 🚀 Installation & Setup
1. **Clone Repo:** `git clone https://github.com/Tangkoan/siemreap_gear.git`
2. **Install:** `composer install && npm install`
3. **Env:** `cp .env.example .env` (Configure your DB)
4. **Migrate & Seed:** `php artisan migrate --seed`
5. **Launch:** `php artisan serve`
